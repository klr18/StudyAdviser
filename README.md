# Мобильное приложение для тайм-менеджмента

## 📖 Описание

StudyAdviser — мобильное Android-приложение для помощи студентам в планировании учебного процесса и организации задач.  
Приложение позволяет создавать учебные задачи, отслеживать прогресс, следить за расписанием и использовать инструменты повышения продуктивности.

## ⚙️ Основной функционал

- Создание и управление учебными задачами
- Формирование академического расписания
- Отслеживание прогресса выполнения задач
- Возможность работы в режиме фокуса с таймером Pomodoro
- Просмотр профиля и статистики
- Получение баллов за выполненные задачи и завершенные сессии Pomodoro

## 🛠 Используемые технологии

- Android  
- Java  
- SQLite

## 🎯 Цель проекта

Разработка мобильного приложения‑помощника для студентов, которое помогает организовать учебный процесс и повысить продуктивность.

## ▶️ Запуск проекта

1. Клонировать репозиторий:  

2. Открыть проект в Android Studio.
   
3. Синхронизировать зависимости Gradle.
   
4. Запустить приложение на эмуляторе или реальном устройстве через кнопку Run.
